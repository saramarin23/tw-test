# Christmas Wishlist

_Coding test for Teamwire using React and Material-UI components._

The task is to create a simple SPA that helps you to write down your wishlist, so Santa can see it and he won't forget what you want for Christmas.

## How to run the app

Clone this repo and run the following command in the folder you have downloaded:

```bash
npm start
```

You can also click [here](https://saramarin23.github.io/tw-test/) to see the demo.
